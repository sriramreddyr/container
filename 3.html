Building a Container

Images can build be automatically by reading the instructions from a Dockerfile. A Dockerfile is a text document that contains all the commands a user could call on the command line to assemble an image.

An image can be build using the command

docker build . -t imageName:tag

podman build commands

Below is the a sample dockerfile to build an image.

FROM ubuntu:latest
RUN apt-get update && apt-get install apache2 -y
RUN rm /var/www/html/index.html
COPY myfile /etc/apache/myfile.conf

podman images ( podman images command) Should list the image built out of the Dockerfile
