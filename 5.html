Pushing an Image to the Registry

Now we have the application built into the image and ready for the deployment.
We need this image to be pushed to the container registry for safe keeping and accessible by others

Now lets try to push this image to the docker registry

Before trying to push the image to the docker, we first need to tag the image to the username and the name space where the image has to go

So lets tag it using podman tag sriram-persentation:latest sriramreddyr/presentation:latest

Now try Podman images, you will see 2 images sriram-persentation & sriramreddyr/presentation. If you check at the imageID of both the images, it will be same

Now we have our image ready and tagged.
podman push sriramreddyr/presentation will push my image to the docker registry. Since we have not give any tag, it would take the default tag as latest and will append to the image name.

Now if someone whats to pull the image they can simply
podman pull sriramreddyr/presentation:latest and get the container image.

And if we were to push this to a private registry, we could also tag the Image with the full repository name.
